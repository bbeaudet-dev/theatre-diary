# AI Collaboration Workspace

This folder is the shared workspace for human + agent coordination.

## Structure

- `general-guidance/`: reusable AI tips, project instructions, coding preferences, and prompts.
- `handoffs/`: work handoffs between agents, key state documents, and continuity notes.
- `prds-specs/`: product requirements, implementation plans, and scoped specs.
- `brainstorming/`: rough ideas, optional notes, and early-stage explorations.
- `templates/`: reusable templates for handoffs, PRDs, and planning docs.

## Suggested usage

- Keep `handoffs/` concise and execution-focused.
- Keep `prds-specs/` decision-focused with clear scope and acceptance criteria.
- Use `brainstorming/` for low-pressure notes that may later graduate into specs.
